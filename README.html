Breccia Parser

This project implements a cursor-based pull parser for efficient reading of Breccian markup. It includes both a basic Breccia cursor that emits native parse states, and a translational cursor that emits X-Breccia, an XML homologue that makes Breccia accessible to the full suite of JAXP, the Java API for XML Processing.
